Color Theory and Concepts

    Color Theory Basics

  • Primary Colors: Red, blue, and yellow—base colors that mix to create others.
  • Secondary Colors: Green, orange, and purple—created by mixing primary colors.
  • Tertiary Colors: Mixes of primary and secondary colors.
  • Warm Colors: Reds, oranges, and yellows—often evoke energy and warmth.
  • Cool Colors: Blues, greens, and purples—often feel calming and professional.

    Color Harmony & Palettes

  • Monochromatic: Variations of a single color (e.g., light blue, medium blue, dark blue).
  • Analogous: Colors next to each other on the color wheel (e.g., blue, teal, green).
  • Complementary: Opposite colors (e.g., blue & orange) create strong contrast.
  • Triadic: Three evenly spaced colors on the wheel (e.g., red, yellow, blue).

RGB Colors
Colors on the internet are described using the RGB color model, regardless of the format you choose: HEX, RGB (r, g, b), or HSL. Here's a brief explanation of each:
  • Hexadecimal (Hex): This format uses a six-digit code to represent colors, with each pair of digits corresponding to red, green, and blue values. For example, #FF5733 is a hex code for a specific shade of orange.
  • RGB (r, g, b): This format specifies the intensity of red, green, and blue components using values from 0 to 255. For example, rgb(255, 87, 51) represents the same shade of orange as the hex code #FF5733.
  • HSL (hue, saturation, lightness): This format describes colors in terms of hue (0-360 degrees on the color wheel), saturation (0-100%), and lightness (0-100%). For example, hsl(14, 100%, 60%) also represents the same shade of orange.
Although these formats look different, they all describe colors based on the RGB color model, which is used for digital displays. The choice of format depends on your preference or the requirements of your project.
